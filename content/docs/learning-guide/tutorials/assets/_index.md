您为项目创建的网格、纹理和动画称为 *源资源*。**Asset Pipeline** 处理源资源以生成运行时优化的 *产品资源*，您可以在 **Open 3D Engine （O3DE）** 项目中使用这些资源。本节中的教程涵盖了使用 **Scene Settings** 工具自定义 Asset Processor 生成产品资源的方式的所有基础知识。
