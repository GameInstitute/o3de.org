**Open 3D Engine (O3DE)** 是一款面向游戏、模拟和多媒体创建者的开源 3D 开发引擎。

O3DE 是由志愿者社区开发的。如果您在文档中找不到想要的内容、发现错误或认为某个主题需要进一步说明，请告诉我们，帮助我们改进文档！

访问 [O3DE 文档 GitHub 仓库](https://github.com/o3de/o3de.org) 提交问题、拉取请求或克隆文档仓库并开始贡献。
