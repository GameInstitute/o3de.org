**Open 3D Engine (O3DE) Frameworks** 是作为 O3DE 核心元素发布的一系列库。框架是底层库，负责为 Gem 和项目代码提供支持，允许用户在平台的通用 SDK 基础上定制自己的 O3DE 项目。
