**Open 3D Engine (O3DE)** 是一个复杂的库套件，旨在与可重复使用的库（**Gem**）和由 SDK 支持的用户创建的项目（**projects**）协同工作。在开发人员指南的这一部分，您将自下而上地了解更多有关 O3DE 的整体架构和设计，从而使核心引擎开发人员能够更好地利用引擎罩下的引擎，诊断调试和加载中的棘手问题，并在牢记核心架构原则的基础上有效地为 O3DE 做出贡献。
