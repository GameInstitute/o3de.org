{{< note >}}
在版本 22.05.0 中，版本控制发布方案发生了变化。二进制安装程序现在与 source 共享一个版本，并且友好名称已被删除。

对于早期版本：源版本使用 `XXXX.Y` 编号模型。二进制版本使用`XX.XX FriendlyName`名称，其中小数位从原始源代码版本向左移动两个值。
{{< /note >}}
