LyTestTools 是旨在简化跨不同环境编写和调试测试的工具。LyTestTools 的一些功能包括环境操作、创建/收集图像和日志以及启动启动器。LyTestTools 所做的很大一部分工作是使用 [Pytest](https://docs.pytest.org/).

LyTestTools 位于`Tools/LyTestTools`.
