您可以在 UI 画布中创建切片的实例。

要实例化切片
在 **UI 编辑器**中，执行以下操作之一：

1. 选择 **New**, **Element from Slice Browser**. 这将显示一个文件浏览器，用于选择您的切片文件。

1. 选择 **New**, **Element from Slice Library**. 这将显示`\UI\Slices\Library`下的所有切片的层次菜单。
