您可以使用 **Actions** 属性在列出的操作之一发生时触发特定事件。为其中一个操作键入字符串。当列出的操作发生时（例如，当游戏玩家开始在元素上暂停时），列出的字符串将作为操作发送。

您可以为以下操作输入字符串：
+ **Hover start**
+ **Hover end**
+ **Pressed**
+ **Release**
