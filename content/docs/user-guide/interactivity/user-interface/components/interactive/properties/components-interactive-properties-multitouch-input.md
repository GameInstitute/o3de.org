**Multi-Touch Input Enabled** 复选框默认处于选中状态。此设置确定组件是否将处理它收到的所有多点触控事件。取消选中该复选框后，此可交互对象将仅处理主要触摸输入事件。

如果未在父 UI Canvas 组件中启用 **Handle multi-touch**，则 O3DE 将忽略此设置。
