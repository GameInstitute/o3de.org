**Open 3D Engine (O3DE)** 通过专用的 [ROS 2 Gem](/docs/user-guide/gems/reference/robotics/ros2) 和其他扩展 Gem 支持机器人模拟。
这些 Gem 提供各种 O3DE 组件，例如传感器、机器人控制和转向、机械臂，并支持从常见格式导入机器人。
