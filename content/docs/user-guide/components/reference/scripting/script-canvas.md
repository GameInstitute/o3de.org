使用 **Script Canvas** 组件将脚本添加到实体。 有关 Script Canvas 组件的信息，请参阅 [使用 Script Canvas 创建游戏和其他行为](/docs/user-guide/scripting/script-canvas/)。
  
## Script Canvas 组件属性

**Script Canvas** 组件具有以下属性：

**Script Canvas Asset**
指定要添加到组件的脚本文件。
