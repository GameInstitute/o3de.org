请参阅 [Robotics](/docs/user-guide/interactivity/robotics)文档，了解有关 O3DE 的机器人仿真功能的更多信息。

ROS 2 Gem 组件可分为以下几类：
-  [核心组件](core), 对于独立于机器人类型运行仿真至关重要。
-  [传感器组件](sensors), 负责创建和发布模拟机器人传感器的数据。
-  [控制组件](control), 由于控制命令而处理移动底座和机器人手臂的移动。
