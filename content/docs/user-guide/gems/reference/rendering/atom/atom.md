Atom Common Features Gem 为 Open 3D Engine 项目添加了 **Atom 渲染器** 以及相关的工具和实用程序，例如 **材质编辑器**。Atom 是一个模块化的、数据驱动的、多线程的渲染器，支持 Forward+ 渲染。

有关更多信息，请参阅 [Atom 指南](/docs/atom-guide).
