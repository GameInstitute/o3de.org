**Atom O3DE Integration** Gem 向 **Open 3D Engine （O3DE）** 添加组件、库和功能以支持 Atom 渲染器。Atom 是一个模块化的、数据驱动的、多线程的渲染器，支持 Forward+ 渲染。

在 O3DE 中使用 Atom 渲染器时，需要 Atom O3DE Integration Gem。

有关更多信息，请参阅 [Atom 指南](/docs/atom-guide).
