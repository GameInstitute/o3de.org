Scripted Entity Tweener Gem 为 Open 3D Engine （O3DE） 项目提供脚本驱动的动画系统。尽管脚本化实体补间对 UI 实体特别有用和强大，但它可以与 O3DE 中的其他实体一起使用。

有关详细信息，请参阅 [脚本化实体补间系统](/docs/user-guide/interactivity/user-interface/animating/tweener-system/).
